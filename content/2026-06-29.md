# Topic 13：还债策略——雪球法 vs 雪崩法

> **本篇目标**：Topic 12 你学会了用「利率」和「用途」两把尺子，看清哪笔债最该先杀。但现实往往不是只有一笔债——你可能同时背着一张信用卡、一笔个人贷款、一笔车贷。**钱有限，先猛攻哪一笔？** 这一篇我们讲两种最经典的打法：**雪崩法（Avalanche）**先打利率最高的，**省钱最多**；**雪球法（Snowball）**先打余额最小的，**心气最足**。读完你将能：①讲清两种方法的数学原理与各自代价；②用互动工具算出在你的债务结构下，雪崩到底比雪球省多少利息；③明白「最优解」和「你真能坚持下来的解」未必是同一个，并据此为自己选边。

---

## 一、先立规矩：两种方法共享的「同一套底盘」

不管雪球还是雪崩，**底盘是完全一样的**，区别只在「额外的钱往哪砸」。这套底盘有三步：

1. **所有债，每月都先还「最低还款额」**——这是不能违约的底线，少还一笔都会吃罚息、伤征信。
2. **把每月预算里超出「所有最低还款额之和」的那部分钱，叫「火力」（额外还款）**，集中砸向**某一笔**目标债务，而不是雨露均沾。
3. **一笔债清零后，它原本的最低还款额不会消失，而是并入「火力」**，去猛攻下一个目标——这就是「滚雪球」效应：火力越打越猛，越往后清得越快。

> 💡 关键洞察：**「集中火力打一笔」永远优于「平均分给每一笔」。** 因为平均分摊会让每笔债都拖得很久、每笔都在持续生息；而集中清掉一笔后，那笔债就**彻底停止计息**，且它的最低还款额转化成对下一笔的增援。两种方法唯一的分歧，就是这第 2 步——**目标债务怎么选**。

- **雪崩法（Avalanche）**：目标 = **当前利率（APR）最高**的那笔。
- **雪球法（Snowball）**：目标 = **当前余额最小**的那笔。

---

## 二、雪崩法：数学上的最优解

雪崩法的逻辑直接承接 Topic 12——既然「还高息债 = 锁定一笔等于其利率的无风险回报」，那**每一块多余的钱，当然要去消灭利率最高的那笔债**，因为它正在以最快的速度膨胀。

把火力优先投给 26% 的信用卡，而不是 5% 的车贷，含义是：你这块钱要么「赚」26%（少付信用卡利息），要么「赚」5%（少付车贷利息）。**理性人显然选 26%。**

$$\text{每一元火力的「回报」} = \text{它所偿还那笔债的年利率}$$

所以雪崩法有一个**可以证明的性质**：在「每月总预算固定」的前提下，**它付出的总利息一定是所有顺序里最少的，还清的总时间通常也最短**。这就是它被称为「数学最优」的原因。

> 📌 一句话记住雪崩法：**哪笔最贵打哪笔。** 它对你的「钱包」最友好。

---

## 三、雪球法：行为金融学的最优解

如果雪崩法在数学上无懈可击，为什么还有那么多理财专家（如美国的 Dave Ramsey）力推雪球法？因为**还债的真正敌人，往往不是利率，而是「半途而废」**。

雪球法先打**余额最小**的那笔债。它的好处不在数学，而在**心理**：

- 余额最小 = **最快被清零**。可能一两个月你就彻底干掉了一整笔债，账户列表里「少了一行」。
- 这个「**第一场胜利**」带来的成就感和掌控感，是巨大的心理燃料。它让你相信「我真的在还清，这事有戏」，从而**更可能坚持到底**。

这不是鸡汤。美国西北大学凯洛格商学院一项被广泛引用的研究发现：在真实还债行为中，**「先清掉小额账户」是预测一个人能否最终还清全部债务的最强信号之一**——比利率、比总额都更能解释谁坚持了下来。原因正是「小胜利→动力→坚持」这条心理链条。

> 💡 一个诚实的类比：减肥时，「先做最难的高强度训练」理论上效率最高，但很多人第三天就放弃了；而「先从能坚持的小目标开始、攒成就感」的人，反而走得更远。**还债是一场马拉松，能跑完的方案，才是好方案。**

> 📌 一句话记住雪球法：**哪笔最小打哪笔。** 它对你的「心气」最友好。

  function readDebts(){

    ids.forEach(function(k){
      var b=parseFloat(document.getElementById('b'+k).value)||0;
      var r=parseFloat(document.getElementById('r'+k).value)||0;

  function simulate(debts, extraPay, strategy){
    // 复制；min 在开始时按初始余额固定（贴近真实分期/账单的做法）
    var d=debts.map(function(x){return {key:x.key,name:x.name,bal:x.bal,mr:x.mr,apr:x.apr,min:Math.min(x.min,x.bal)};});
    var totalMin=0; d.forEach(function(x){totalMin+=x.min;});
    var budget=totalMin+extraPay;
    var months=0, interest=0, cap=1200, order=[];
    while(d.some(function(x){return x.bal>0.005;}) && months<cap){
      months++;
      // 计息
      d.forEach(function(x){ if(x.bal>0){ var i=x.bal*x.mr; x.bal+=i; interest+=i; } });
      var pool=budget;
      // 先还最低（已清零的债不再占用，其最低额自动并入火力）
      d.forEach(function(x){ if(x.bal>0){ var p=Math.min(x.min,x.bal); x.bal-=p; pool-=p; } });

      // 选目标并集中火力
      function pick(){
        var act=d.filter(function(x){return x.bal>0.005;});
        if(!act.length) return null;
        return strategy==='avalanche'
          ? act.reduce(function(a,b){return b.apr>a.apr? b:a;})
          : act.reduce(function(a,b){return b.bal<a.bal? b:a;});
      }
      var t=pick();
      while(pool>0.005 && t){
        var pay=Math.min(pool,t.bal);
        t.bal-=pay; pool-=pay;
        if(t.bal<=0.005){ t.bal=0; t=pick(); }
        else break;
      }
      // 记录本月清零的债
      d.forEach(function(x){
        if(x.bal<=0.005 && order.indexOf(x.key)<0){ order.push(x.key); x._cleared=months; }

## 五、到底选哪个？一张决策清单

没有放之四海皆准的答案，但可以对号入座：

| 你的情况 | 更适合 | 理由 |
|---|---|---|
| 各笔债**利率差距很大**（如 26% 信用卡 + 3% 房贷） | **雪崩** | 省下的利息可观，理性收益压倒心理收益 |
| 各笔债**利率差不多** | **雪球** | 反正省不了多少，不如要那份动力 |
| 你**曾经多次半途放弃**、需要正反馈 | **雪球** | 「第一场胜利」是你最缺的燃料 |
| 你**自律性强、对数字敏感、目标导向** | **雪崩** | 你扛得住「最贵那笔拖最久」的心理煎熬 |
| 有一两笔**小额债**烦得你睡不着 | **雪球**（或混合） | 先清掉小债换清净，再回头雪崩攻高息 |

> 💡 **混合策略（很实用）**：先用雪球法快速清掉一两笔最小的债，拿到「我能行」的成就感和现金流喘息，**然后切换到雪崩法**集中干掉剩下的高息大债。这等于「先点火，再追求效率」，兼顾了心理与数学。很多理财顾问实际推荐的就是这种打法。

> ⚠️ **两个前提别忘了**（来自前几篇）：①动手还债前，**应急金（Topic 11）得先到位**——否则一遇急事又得刷卡借债，前功尽弃；②**别一边高息滚债一边盲目投资**（Topic 12）——清掉 26% 的债，就是你能买到的最佳「投资」。

## 📝 本篇小结

- **两种方法共享同一套底盘**：所有债先还最低 → 把「额外火力」集中砸向一笔 → 清掉一笔后其最低额并入火力（滚雪球）。唯一分歧是「先攻哪笔」。
- **雪崩法（哪笔最贵打哪笔）= 数学最优**：在固定预算下总利息最少、通常也最快还清，因为每一元火力都去消灭膨胀最快的债。
- **雪球法（哪笔最小打哪笔）= 行为最优**：先清小债带来「第一场胜利」，用成就感喂养坚持。研究显示「先清小额账户」是预测能否最终还清的最强信号之一。
- **取舍本质**：雪球用「多付一点利息」换「更强的坚持力」。利率差距大→雪崩更划算；利率接近或你需要动力→雪球更明智；**混合法**（先雪球点火、再雪崩提效）兼顾两者。
- **前提**：先备好应急金、别一边滚高息债一边盲投。能让你**坚持到债务清零**的方案，才是最好的方案。
